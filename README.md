# libvoicefxv2

Voice changer AML mod untuk SA-MP Android (MoNetLoader 3.6.0).

## Install
1. `libvoicefx.so` → `/Android/data/com.sampmobilerp.game/files/AML/mods/`
2. `voicechanger.lua` → folder MoNetLoader scripts

## Commands
- `/vfx` — toggle ON/OFF
- `/vfp 0.5` — set pitch (0.3–3.0)
- `/vfs` — status
